  1. What is jQuery?
    2. Ans:-
    It is a small, fast and lightweight JavaScript library.It is also platform-independent.It means "write less do more".It also simplifies AJAX call and DOM manipulation.
  2. How to Apply CSS Using JQuery, How to Add Class and Remove Class in Jquery , JQuery Animation?
    3. Ans:-
    For applying css using jQuery
    Syntax:
    $(selector).css(property)
    For Adding a class
    Syntax:
    $('selector').addClass(class_name);
    For Removing a class
    Syntax:
    $('selector').removeClass(class_name);
    For jQuery animation
    Syntax:
    $(selector).animate({params}, speed, callback);
  3. How to create slider with animation?
    4. Ans:-
    We can create slider with animation using this syntax:
    $("selector").animate({params}, speed, callback);
