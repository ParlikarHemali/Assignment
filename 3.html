  1. What are the new tags added in HTML5?
    • Ans:-

      The new tags added in HTML5 are as follow:

      1. Article:It is one of the new sectioning element in HTML5. The HTML < article > tag is used to represent an article.
      2. Aside:It is used to describe the main object of the web page in a shorter way like a highlighter.
      3. Audio:It is used to insert audio in webpage.
      4. Canvas:It is used to draw graphics on a web page using javascript.
      5. Datalist:It is used to provide autocomplete feature in HTML files.
      6. Details:It is used for the content/information which is initially hidden but could be displayed if the user wishes to see it.
      7. Embed:It is used for embedding external applications which are generally multimedia content like audio or video into an HTML document.
      8. Figure:It is used to add self-contained content like illustrations, diagrams, photos, or codes listing in a document.
      9. Footer:It is used to define a footer of HTML document.
      10. Header:It contains information related to the title and heading of the related content.
      11. Hgroup:It stands for heading group and is used to group the heading elements.
      12. Keygen:It is used to specify a key-pair generator field in a form.
      13. Mark:It is used to define the marked text.
      14. Meter:It is used to define the scale for measurement in a well-defined range and also supports a fractional value.
      15. Nav:It s used for declaring the navigational section in HTML documents.
      16. Output:It is used to represent the result of a calculation performed by the client-side script such as JavaScript.
      17. Progress: It is used to represent the progress of a task.
      18. Ruby:It is used to specify the ruby annotation which is a small text, attached with the main text to specify the meaning of the main text.
      19. Section:It defines the section of documents such as chapters, headers, footers, or any other sections.
      20. Time:It is used to display the human-readable date/time.
      21. Wbr:It stands for word break opportunity and is used to define the position within the text which is treated as a line break by the browser.
      22. Video:It is used to embed video content in a document, such as a movie clip or other video streams.
  2. How to embed audio and video in a webpage?
    • Ans:-
      1. To embed audio in HTML,we use < audio > tag.Before HTML5 we can't add audio in web pages. After the release of HTML5 it is possible to add audio in web pages.
        Syntax: < audio > < source src="file_name"> < /audio >

      2. To embed video in HTML, we use the < video > tag.
        Syntax: < video > < source src="file_name"> < /video >
  3. Semantic element in HTML5?
    • Ans:-

      Semantic elements have meaningful name:s which tells about type of content. For example header, footer, table, … etc. HTML5 introduces many semantic elements as mentioned below:

      1. Article: It contains independent content which doesnt require any other context.
      2. Aside: It is used to place content in a sidebar i.e. aside the existing content. It is related to surrounding content.
      3. Details and Summary: “details” defines additional details that the user can hide or view. “summary” defines a visible heading for a “details” element.
      4. Figure and Figcaption: These are used to add an image in a web page with small description.
      5. Header: As the name suggests, it is for the header of a section introductory of a page. There can be multiple headers on a page.
      6. Footer: Footer located at the bottom of any article or document, they can contain contact details, copyright information etc. There can be multiple footers on a page.
      7. Main: It defines the main content of the document. The content inside main tag should be unique.
      8. Section: A page can be split into sections like Introduction, Contact Information, Details etc and each of these sections can be in a different section tag.
      9. nav: It is used to define a set of navigation links in the form of navigation bar or nav menu.
      10. Mark: It is used to highlight the text.
  4. Canvas and SVG tags
    • Ans:-

      The < canvas > tag in HTML is used to draw graphics on a web page using JavaScript. It can be used to draw paths, boxes, texts, gradients, and adding images. By default, it does not contain borders and text.
      The < canvas > tag is new in HTML5.
      Syntax: < canvas id = "script" > Contents... < /canvas >


      The < svg > tag is used as a container for SVG graphics. SVG (Scalable Vector Graphics) is a language for two-dimensional graphics based on XML with support for animation and interactivity. To draw images, it uses simple geometrical figures (circle, lines, polygon, etc.).
